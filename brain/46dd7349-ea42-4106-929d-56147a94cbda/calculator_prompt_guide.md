# 🪄 AI 요청용 초고성능 '마법의 프롬프트' 가이드북

이 문서는 사용자가 향후 **ChatGPT, Claude, Gemini** 등의 AI 모델에게 자신만의 업무용 계산기를 제작해 달라고 요청할 때 바로 복사하여 사용할 수 있는 **최고 효율의 프롬프트 템플릿**입니다.

이 프롬프트는 AI가 코드를 생략하지 않고, 최고 수준의 UI/UX 디자인(글래스모피즘)과 완벽한 회계 연산(부가세 4단 콤보) 기능이 포함된 단일 HTML 파일을 생성하도록 정밀하게 설계되었습니다.

---

## 📋 [복사용] 업무용 계산기 제작 요청 프롬프트

아래의 박스 안에 있는 내용 전체를 그대로 복사하여 AI 채팅창에 입력하시면 됩니다.

```text
너는 세계 최고 수준의 프론트엔드 UI/UX 개발자이자 금융/회계 솔루션 설계자야.
사무실에서 재무, 회계, 세무 업무를 처리할 때 눈이 즐겁고 사용하기 극도로 편리한 "초프리미엄 업무용 계산기"를 단일 HTML 파일로 제작해 줘.

다음 요구사항을 단 한 줄의 코드 생략도 없이 완벽하고 구체적인 코드로 작성해 줘.

### 1. 디자인 및 테마 요구사항 (Rich Aesthetics & Premium UX)
- **비주얼 컨셉**: 투명한 유리 질감을 살린 '글래스모피즘(Glassmorphism)'과 몽환적인 파스텔 네온 그라데이션 배경을 적용해 줘. (`backdrop-filter: blur(20px)` 활용)
- **다크/라이트 테마**: 화면 상단에 테마 전환 토글 버튼을 배치하여, 사용자가 밤낮에 맞춰 눈이 편한 다크 모드와 화사한 라이트 모드를 자유롭게 전환할 수 있게 해 줘.
- **애니메이션 피드백**: 계산기 버튼을 마우스로 가리키면 살짝 뜨고(hover), 누르면 쫀득하게 안으로 들어가는 미세한 클릭 애니메이션(active scale)과 그림자 변화를 적용해 줘.
- **반응형 웹**: PC 브라우저뿐만 아니라 태블릿, 모바일 기기에서도 화면 비율이 깨지지 않고 완벽하게 정렬되도록 CSS Flexbox와 Grid를 활용해 설계해 줘.

### 2. 계산 기능 및 회계 맞춤형 로직
- **기본 사칙연산**: 더하기(+), 빼기(-), 곱하기(×), 나누기(÷), 퍼센트(%), 소수점(.), 지우기(⌫), 전체초기화(C) 기능.
- **3자리 단위 쉼표(,)**: 회계 장부를 볼 때처럼 입력하는 도중 및 결과값 모두에 3자리마다 자동으로 쉼표가 찍히도록 해 줘 (예: 1,000,000). 계산식 내부 연산 시에는 쉼표를 제거하고 처리해야 해.
- **00 입력 버튼**: 금액 입력 편의를 위해 '0' 외에도 '00' 버튼을 추가해 줘.
- **부가세(VAT) 특화 4단 연산 버튼**:
  1. [VAT +10%]: 현재 입력값(공급가)에 10% 부가세를 더한 합계금액을 계산 (값 * 1.1)
  2. [VAT -10%]: 현재 입력값(합계금액)에서 10% 부가세를 제한 순수 공급가액을 계산 (값 / 1.1)
  3. [VAT 세액]: 현재 입력값(공급가) 기준 10%의 순수 세금 액수만 추출 (값 * 0.1)
  4. [VAT 내부세]: 부가세가 포함된 합계금액 속에 숨어있는 순수 부가세 세액만 분리 (값 / 11)
  * 모든 부가세 연산 결과는 정수형으로 깔끔하게 반올림 처리(Math.round) 해 줘.
- **부동소수점 오차 방지**: JavaScript 특유의 소수점 연산 오차(예: 0.1+0.2=0.30000000000000004)가 발생하지 않도록 정밀도를 조정해 줘 (`.toFixed(10)` 등 사용).

### 3. 업무 편의 기능
- **결과값 복사 버튼**: 계산기 디스플레이 우측 상단에 복사(📋) 버튼을 제공해 줘. 누르면 결과값이 클립보드에 복사되고 화면 아래에 세련된 "복사 완료!" 토스트 메시지가 나타나게 해 줘.
- **계산 기록(History) 보관판**: 계산기 오른쪽에 이전에 계산했던 수식과 결과들이 기록 리스트로 차곡차곡 쌓이게 해 줘.
  - 기록을 클릭하면 그 값이 계산기 입력창으로 다시 로드되어 연속 계산이 가능하게 해 줘.
  - 브라우저를 껐다 켜도 내역이 지워지지 않도록 `localStorage`에 자동 보관 및 불러오기를 구현해 줘.
  - [모두 비우기] 버튼을 통해 기록을 한 번에 지울 수 있게 해 줘.

### 4. 코드 작성 규칙 (초보자 배려)
- **단일 파일 배포**: HTML, CSS, JavaScript를 다른 외부 파일 분리 없이 오직 한 개의 `index.html` 파일 내부에 모두 작성해 줘. 더블 클릭만으로 브라우저에서 실행되도록 할 거야.
- **코드 생략 절대 금지**: `// ... 기존 코드와 동일 ...` 이나 `// 로직 구현 생략` 같은 축약 기호는 절대 사용하지 말고, 전체 코드를 100% 다 적어 줘. 바로 복사해서 덮어쓸 수 있어야 해.
- **상세한 한글 주석**: 각 기능과 스타일 속성, JavaScript 함수의 모든 라인에 이 코드가 무엇을 의미하는지 초보자가 쉽게 읽고 배울 수 있도록 친절한 한글 주석을 풍부하게 작성해 줘.
```

## 💡 AI 프롬프트 사용 꿀팁!

1. **원하는 컬러 변경하기**:
   - 위 프롬프트에서 `비주얼 컨셉` 부분에 `"민트색과 차콜색 테마로 설계해 줘"`, `"밝고 화사한 연분홍 벚꽃 그라데이션을 적용해 줘"`와 같이 원하는 색상을 명시하면 AI가 해당 색상에 딱 맞추어 CSS 코드를 변경해 줍니다.
2. **원하는 기능 추가하기**:
   - `계산 기능` 항목 밑에 `[달러 환전 버튼]: 현재 결과값을 오늘 환율(예: 1350원)로 나누어 달러($)로 환산해 표시하는 기능` 같은 커스텀 비즈니스 규칙을 추가해 보세요. AI가 훌륭하게 코드를 덧붙여 줍니다.
3. **오류 해결 요청하기**:
   - 혹시 AI가 준 코드가 작동하지 않거나 특정 버튼을 누를 때 에러가 난다면, 에러 내용이나 현상을 복사해서 **"이러이러한 오류가 나는데, 이를 수정한 전체 코드를 다시 작성해 줘"**라고 요청하시면 됩니다.
